Introduction

Python is a high-level, interpreted programming language.

It supports multiple programming paradigms and is widely used for web, automation, and data analysis.

print("Hello, Python!")

It has a simple syntax that's easy to learn.

  • Easy to learn
  • Interpreted
  • Dynamic Typing
Data Types

Python has various built-in data types.

These include numbers, strings, lists, tuples, sets, and dictionaries.

x = 42 # Integer name = "Alice" # String
  • List
  • Tuple
  • Set
  • Dict
Control Structures

Control flow is governed by if, for, and while statements.

Python uses indentation instead of braces.

if x > 0:\n print("Positive")

Loops are useful for repeating code.

Functions

Functions help break code into reusable blocks.

You define a function using the def keyword.

def greet():\n print("Hello!")

Functions can take parameters and return values.

Modules and Packages

Modules help organize code across multiple files.

Packages are directories containing modules.

import math from datetime import date

You can install external packages with pip.

  • math
  • datetime
  • random
Exception Handling

Python provides a robust way to handle errors using try-except blocks.

try:\n x = 10 / 0\nexcept ZeroDivisionError:\n print("Cannot divide by zero!")

This prevents the program from crashing unexpectedly.

File Handling

Python makes it easy to read and write files using the built-in open() function.

with open("example.txt", "r") as file:\n content = file.read()

The 'with' keyword ensures the file is properly closed after use.

List Comprehensions

List comprehensions offer a concise way to create lists.

squares = [x*x for x in range(10)]

They're cleaner and more Pythonic than using loops.

Object-Oriented Programming

Python supports OOP principles such as encapsulation, inheritance, and polymorphism.

class Person:\n def __init__(self, name):\n self.name = name\n def greet(self):\n print(f"Hello, {self.name}")

This approach helps organize and scale your code efficiently.

Decorators

Decorators are functions that modify the behavior of other functions.

def decorator(fn):\n def wrapper():\n print("Before function call")\n fn()\n print("After function call")\n return wrapper @decorator\ndef say_hello():\n print("Hello!")

This allows enhanced functionality without changing original code structure.
